# クイックスタート

セットアップ完了後、最初のクエリを送るまでの手順です。

## Step 1 — オーケストレーター起動

```bash
uvicorn src.orchestrator.server:app --reload --port 8000
```

## Step 2 — GUI 起動

```bash
python scripts/launch_gui.py
```

ブラウザで `http://localhost:7860` を開きます。

## Step 3 — セットアップ確認

**📖 ガイド** タブを開くと「セットアップウィザード」が状態を自動チェックします。

- ✅ プロバイダー設定
- ✅ APIキー
- ✅ オーケストレーター接続

すべて ✅ になれば準備完了です。

## Step 4 — はじめてのクエリ

**💬 チャット** タブを開き、サンプルプロンプトボタンをクリックするか、
直接入力してください。

```
Python で二分探索を実装してください
FAISSとは何ですか？
このコードのバグを直してください: [コードを貼り付け]
```

## レスポンスの読み方

```
┌─────────────────────────────────────┐
│ 回答テキスト...                       │
│                                      │
│ モデル: claude-sonnet-4-20250514     │ ← 実際に使われたモデル
│ レイテンシ: 1234 ms                  │
│ ソース: [github.com/...] score=0.87  │ ← RAG が参照した URL
└─────────────────────────────────────┘
```

!!! tip "オーケストレーター未起動でも動作します"
    未起動の場合は `[MOCK]` 応答が返ります。機能を試したい場合は `--mock` モードとして使えます。
